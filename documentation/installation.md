# Installation 

***Ensure Java 17+ and Gradle Build Tools are installed and set up before proceeding***
1. Clone the GitHub repo
2. Open the command terminal in the “project-poplar” folder
3. Run the command “./gradlew run”

The game window should pop up!

# Testing
1. Open the command terminal in the “project-poplar” folder
2. Run the command “./gradlew test”

This runs the automated tests.

# Game Controls
1. Start the game by pressing "start"
2. Complete the series of wellness tasks, then select and plant the flower
3. View your garden by pressing "garden"
4. View your past entries by pressing "entries"
   
*For testing purposes, keyboard shortcuts are provided to advance to the next day so new entries can be added. A confirmation message appears in the terminal window when successfully executed*
6. "F1": advance to the next day
7. "F2": to plant 14 flowers in the garden
8. "F3": to plant 15 flowers in the garden
7. "F12": to clear all garden and entry data
